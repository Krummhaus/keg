# DuckDuckGo searchin via w3m browser

Simply put your function to `.bashrc` and then `source ~/.bashrc`.  
```bash
echo 'duit(){ w3m https://lite.duckduckgo.com/html/?q="$*"; }' >> ~/.bashrc
```
